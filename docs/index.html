Typeahead

The UI component of typeahead.js is available as a jQuery plugin. It's responsible for rendering suggestions and handling DOM interactions.

This project is actually Bootstrap 4 version of twitter typeahead.js which already has everything we need to build simple typeahead. For complete documentation and example, please refer to the documentation of twitter typeahead.js it self.

Important! bloodhound is not included. If you need it, please include it manually.
